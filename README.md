# Cloudflare

Jednoduchá webová stránka o službě **Cloudflare** a jejích funkcích.

🔗 Web: https://og4575.github.io/cloudflare/

## O projektu

Tento projekt vysvětluje:

- Co je Cloudflare
- Jak funguje
- K čemu slouží
- Jaké služby nabízí

## Funkce Cloudflare

- **CDN** – rychlejší načítání webů  
- **DDoS ochrana** – ochrana proti útokům  
- **WAF** – zabezpečení webových aplikací
